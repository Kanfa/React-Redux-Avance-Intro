# Starter React V16 + Redux

+ ```npm install```
+ ```npm start```

Si vous souhaitez générer votre app vous même il vous faudra juste node avec une version >=6 et npm puis :

+ ```npm install react-create-app```

+ ```npm create-react-app votre-nom-d-app```
+ ```cd votre-nom-d-app ```
+ ```npm start```
